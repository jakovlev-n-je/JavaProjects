Задание:
Реализовать консольную программу, которая имитирует взаимодействие с неким хранилищем в оперативной памяти. Хранилище содержит словарь на базе косого дерева и хранит пары Ключ - Значение. Интерфейс консольного приложения в цикле ожидает и обрабатывает вводимые пользователем команды
Необходимо реализовать следующие команды:
set <ключ> <значение> - добавление / перезапись значения по ключу
get <ключ> - получение значения по ключу
del <ключ> - удаление ключа и соответствующего значения из хранилища
