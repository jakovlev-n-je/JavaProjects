Задание:
Реализовать префиксное дерево, которое будет хранить данные о введенных словах и числе их вводов. Реализовать простой интерфейс, состоящий из поля ввода слова и кнопки. При нажатии на кнопку введённый в поле текст (если он не пустой) добавляется в словарь со значением 1, или соответствующее ему значение увеличивается на единицу, если слово уже содержалось в структуре. При введении символов в текстовое поле предлагать список возможных завершений слова на основе данных из структуры. При этом предложенный список должен быть отсортирован в порядке убывания частоты ввода слов, а его максимальная длина должна быть ограничена любым заранее определенным целочисленным значением
